# SRCC Tech Resource Library

Use this folder as the file drop area for SOPs, references, vendor docs, and supporting assets.

## Folder map

- `resources/sops/`: finalized SOPs and process docs
- `resources/runbooks/`: troubleshooting and incident response steps
- `resources/references/`: reference manuals, wiring notes, standards, cheat sheets
- `resources/vendor-docs/`: manufacturer PDFs, release notes, product docs
- `resources/diagrams/`: rack maps, signal flow diagrams, and room layouts
- `resources/templates/`: reusable templates for SOPs, checklists, and handoff docs
- `resources/handoff/`: temporary handoff bundles from projects
- `resources/archive/`: old or superseded resources

## Suggested naming

Use date-first names when possible:

- `YYYY-MM-DD-topic.ext`
- `YYYY-MM-DD-system-sop-v1.md`

## Notes

- You can paste any file type here (`.md`, `.pdf`, `.docx`, `.xlsx`, images, etc.).
- Keep one source of truth per document and move old versions to `archive/`.
